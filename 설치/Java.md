# Java
```
version : JDK12.0.2

대부분 1.8이나 11을 많이 쓴다.
버전이 높은 것은 아래 버전을 사용할 수있지만, 버전이 낮은 것에선 위에 버전을 사용할 수 없다.

1. 설치 후 마지막 화면에서 close

2. C:\Program Files\Java 생성 확인

내 PC 화면에서 오른쪽클릭 -> 속성 -> 고급시스템 설정 -> 환경변수 -> 시스템 변수에서 새로 만들기 클릭
-> 변수이름 아무거나(ex. JAVA_HOME) -> JDK가 설치된 폴더 경로 들어가기(ex. C:\program Files\Java\jdk-12.0.2) -> 변수 값에 그 경로 주소 적기
-> 시스템 변수이름 Path 편집 -> 새로만들기 -> '%변수이름%\bin\' 입력(ex. %JAVA_HOME%\bin\) -> 창 전부 "확인 버튼"으로 끄기

3. cmd에 "javac"와 "java-version" 하나씩 입력해서 제대로 설치 됐는지 확인하기
```
