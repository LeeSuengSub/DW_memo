# MySQL
```
MySQL이 C++로 만들어 졌기 때문에 C++이 설치가 안되어있으면 설치해야 한다.

https://www.mysql.com/downloads/
    -> MySQL Community (GPL) Downloads
    -> MySQL Community Server
    -> Recommended Download
    -> 웹버전 말고 용량 큰 버전 설치

디폴트 port : 3306
비밀번호 입력
서비스 이름도 디폴트 MYSQL80 으로 설치
connect to server status running으로 떠있는지 확인 : 각 PC가 서버가 된 상태
밑에 비밀번호 입력하고 체크하기
```

## 설치 확인하는 방법
```
CMD에 netstat-a 입력
주소 3306 확인
윈도우 검색창에 서비스 검색
MySQL80 서비스 "실행중" 확인
```
