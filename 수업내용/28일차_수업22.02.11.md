- 폴더에 숫자가 자동으로 더해져서 생성됨.

```
div.item${$}*10
```
- 중복되는 코드들을 줄이는 방법



overflow: 컨텐츠가 너무 커서 요소내에 모두 보여주기 힘들 때 사용.
```
overflow: visible(기본값)  
          hidden(넘치는 부분 자름)  
          scroll(넘치는 부분에 맞게 스크롤이 생김.)  
```
