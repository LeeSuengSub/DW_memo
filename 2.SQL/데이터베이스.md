# 데이터베이스
---
```
데이터베이스 종류는 여러개이나 문법은 동일하다.
-> 오라클, MySQL, MariaDB

특징
- 오라클 : 국가기관이나 공공기관이 많이 사용
- MySQL : 스타트업, 중견기업이 많이 사용

요즘은 국가기관도 오라클 대신 티맥스에서 만든 TIBERO를 사용.
```

## 장점
---
```
데이터베이스를 잘 배우면 어느 분야에 가든 도움이 된다.
-> 웹, 앱, IoT, 데이터분석분야, AI분야
```

## 데이터베이스 구성도
---
```
MySQL -> DataBase -> Table -> Data
=> 여러 데이터베이스가 존재
   데이터베이스에 여러 테이블이 존재
   테이블에 데이터가 존재

대부분의 테이블에는 다 관계가 있다.
```

## 데이터베이스 설명
---
```
데이터베이스에는 두가지 종류가 있다.

- 관계형 데이터베이스
- 비관계형 데이터베이스

관계형 데이터베이스 : 웹, 앱 개발에 사용
비관계형 데이터베이스 : IoT, AI개발에 사용

컴퓨터는 문자, 숫자 구분을 못하므로 데이터 타입을 입력해야 하낟,
숫자 : int
문자 : varcahr

PK(Primary Key) : 테이블의 고유 컬럼으로 고유한 데이터를 가지고 있다.
-> ex) 사원번호, 주민등록번호
컬럼(속성) : 테이블을 구성하는 각각의 열에 위치한 정보.
데이터 : 테이블에 최종으로 저장된 데이터들.

데이터베이스는 커밋(commit)을 해야 데이터가 쌓인다.
```
